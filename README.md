# 🏡 StayNest – Property Rental Platform

StayNest is a full-stack property rental platform that allows hosts to list and manage properties while guests can browse homes, view details, manage favourites, and discover comfortable stays. The platform is built using Node.js, Express.js, MongoDB, EJS, and Tailwind CSS following the MVC architecture.

## 🌐 Live Demo

🔗 **Live Website:** https://stay-nest-f4wf.vercel.app/

## 🚀 Features

### Authentication & Authorization

* User Signup and Login
* Secure Session-Based Authentication
* Logout Functionality
* Protected Routes

### Property Management

* Add New Property Listings
* Edit Property Details
* Delete Properties
* View All Available Properties
* Property Details Page

### User Features

* Browse Properties
* Add Properties to Favourites
* Remove Properties from Favourites
* View Favourite Properties
* Access Bookings Page

### Media Upload

* Upload Property Images
* Dynamic Image Rendering using Multer

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Shwetaak15/StayNest.git
```

### Navigate to Project

```bash
cd StayNest
```

### Install Dependencies

```bash
npm install
```

### Start Application

```bash
npm start
```

### Open in Browser

```text
http://localhost:3003
```
